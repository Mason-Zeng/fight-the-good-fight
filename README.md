# fight-the-good-fight
Do everything we can to fight sin.

# What is this?
It is a Chrome browser extension that reminds us not to give into sin. In just one click or keyboard command, this popup will remind us how we must abstain from sin.

# How to use?
1. Go to chrome://extensions/ and make sure Developer Mode is turned on.
2. Press the Load unpacked button and upload the source folder.
3. To see the popup, make sure you are using Chrome.
4. Press Ctrl+Shift+Y or pin the extension and click on it.
If you need help uploading the extension, watch this video starting from 2:00: https://youtu.be/xiT8c8M1OIw

# Features
- The popup currently cycles through 5 hardcoded encouragement texts. You can input your own by clicking the Add Text button.
- You can unmute the video by simply hovering your mouse over the popup.
